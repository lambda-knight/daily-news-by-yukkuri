<h1>スペインニュース 2026年7月1日</h1>
<h2>国際情勢（スペイン視点）</h2>
<h3>ベネズエラ地震、悲しみから怒りへ</h3>
<ul>
<li>El Pais Internacional は、ベネズエラ地震から5日が過ぎ、被災地で住民と当局の緊張が高まっていると報じた。</li>
<li>別記事では、ラ・グアイラに入ったスペインの救助隊が、崩落した建物の中で「聞こえる人は2回たたいて」と呼びかけながら捜索している様子を伝えている。</li>
<li>昨日までの「強制送還者147人が被災した」という悲劇は、今日は「救助の遅れ」「当局への怒り」「国際支援の必要性」という続報に変わった。</li>
<li>スペインではベネズエラ系住民が多く、ラテンアメリカの災害は遠い外国のニュースではない。家族、移民、旧植民地との歴史が重なり、欧州の他国より感情的に近く報じられる。</li>
</ul>
<h3>南アフリカの反移民デモをスペインが読む意味</h3>
<ul>
<li>El Pais Internacional は、南アフリカで外国人排斥を掲げる団体の最後通告後、数千人が移民反対デモを行ったと報じた。</li>
<li>報道では、失業、犯罪、公共サービス悪化の責任を外国人に押しつける構図が強調されている。</li>
<li>これはスペイン国内の移民正規化論争とも直結する。スペインも労働力不足と移民受け入れの間で揺れており、南アフリカのニュースは「他国の暴発」としてだけでなく、自国への警告として読まれる。</li>
<li>日本が移民問題を制度・治安・技能実習の話として語りがちなのに対し、スペインでは街頭政治、労働市場、人種差別が同時に議論になる。</li>
</ul>
<h3>日中緊張とラテンアメリカ分断は続報扱い</h3>
<ul>
<li>El Pais は日中緊張について、小泉進次郎防衛相が「日本は軍国主義を復活させていない」と反論した件を引き続き掲載している。</li>
<li>フランコ独裁を経験したスペインでは、軍事化や権威主義への警戒感が強く、日中関係も「民主主義と権威主義」の枠で読まれやすい。</li>
<li>「ラテンアメリカは半分に割れている」という論評では、急進的な提案が社会階層をまたいで支持を広げていると指摘された。</li>
<li>スペインにとってラテンアメリカは言語と家族でつながる圏域であり、政治的分断も「向こうの話」ではなく、スペイン国内の移民社会にも跳ね返る。</li>
</ul>
<h2>スペイン国内ニュース</h2>
<h3>移民大量正規化、EU法との衝突を最高裁が検討</h3>
<ul>
<li>El Pais Espana は、スペイン最高裁が移民正規化手続きについて、EU法に抵触するか欧州司法に照会する可能性を検討していると報じた。</li>
<li>El Mundo は政府側の説明として、「これはスペイン国内だけの居住・就労許可で、欧州全体に影響しない」と伝えている。</li>
<li>右派系の州政府や自治体は、正規化の効果がすぐ出れば現場の負担が増えるとして停止を求めている。</li>
<li>スペインらしいのは、移民を単なる労働力としてではなく、EU、地方自治、裁判所、街頭世論の複合問題として扱う点だ。日本の技能実習制度より政治の表面に出やすい。</li>
</ul>
<h3>オペラシオン・キッチン、元内相の関与を検察が主張</h3>
<ul>
<li>El Pais Espana は、検察が元内相ホルヘ・フェルナンデス・ディアス氏をオペラシオン・キッチンの中心に位置づけたと報じた。</li>
<li>検察は、ラホイ政権時代の元PP幹部の情報を警察が違法に探った疑惑について、元内相が「統制していた」と最終弁論で主張している。</li>
<li>右派PPの過去の汚職・警察利用疑惑は、左派PSOE周辺の疑惑と並び、スペイン政治全体への不信を強めている。</li>
<li>日本なら「政治と警察の距離」として慎重に報じられる話だが、スペインでは政党対立の中で真正面から劇場化する。</li>
</ul>
<h3>モネデロ氏のセクハラ処分と左派の自己矛盾</h3>
<ul>
<li>El Pais Espana は、ポデモス創設者の一人フアン・カルロス・モネデロ氏が、性差別的ハラスメントでマドリード・コンプルテンセ大学から1年間の職務停止処分を受けたと報じた。</li>
<li>El Mundo Cultura も、イニゴ・エレホン氏をめぐる論考で、左派男性の感情教育や自己点検の不足を扱っている。</li>
<li>フェミニズムを強く掲げてきたスペイン左派にとって、内部から出るハラスメント疑惑は政治的に重い。</li>
<li>スペイン社会では、被害告発が党内問題にとどまらず、大学、メディア、文化欄まで広がって議論される。</li>
</ul>
<h3>アンダルシア州、Voxがモレノ氏に初回投票で拒否</h3>
<ul>
<li>El Mundo は、アンダルシア州首相再任を目指すPPのモレノ氏に対し、極右Voxが初回投票で支持を拒んだと報じた。</li>
<li>10月の再選挙の可能性がちらつく中、左派のマリア・ヘスス・モンテロ氏も野党トップとして応酬した。</li>
<li>アンダルシアはフラメンコと観光の土地である一方、失業、農業、移民、地域アイデンティティが絡む政治の激戦地でもある。</li>
<li>日本でいえば地方議会の首長選が国政の代理戦争になる構図に近いが、スペインでは地域の誇りがより前面に出る。</li>
</ul>
<h2>文化・芸能ニュース</h2>
<h3>ムッソリーニ評伝完結、ファシズムの最後のページ</h3>
<ul>
<li>El Pais Cultura は、アントニオ・スクラティ氏のムッソリーニ評伝シリーズ「M」最終巻がスペインで刊行されたと報じた。</li>
<li>著者は「ファシズムの最後のページこそ最悪だ」と語り、独裁の残響が現代政治にどう残るかを問い直している。</li>
<li>フランコ独裁の記憶を持つスペインで、ムッソリーニ評伝が文化欄の大きな話題になることには意味がある。</li>
<li>日本の戦争記憶論争と同じく、過去の独裁や軍国主義をどう語るかは、現在の政治姿勢そのものを映す。</li>
</ul>
<h3>セビリアでソローリャ作品が歩道から消える</h3>
<ul>
<li>El Pais Cultura は、ある家族が海辺の休暇へ持っていくつもりだったソローリャの絵を、車へ積み込む途中で歩道に立てかけ、そのまま消えたと報じた。</li>
<li>警察は窃盗の可能性を調べている。ソローリャはスペイン近代絵画の重要画家で、光と海辺の表現で知られる。</li>
<li>美術品が美術館ではなく家族の生活動線の中にあるところに、スペインの古い家族資産と文化財意識の不思議な近さが見える。</li>
<li>日本なら掛け軸や茶道具を一時的に外へ出したまま紛失したような話で、文化財の重みと日常のうっかりが衝突している。</li>
</ul>
<h3>コッポラ『メガロポリス』の失敗を描くメガドック</h3>
<ul>
<li>El Pais Cultura と El Mundo Cultura は、フランシス・フォード・コッポラの大作『メガロポリス』制作過程を追ったドキュメンタリー『Megadoc』を扱っている。</li>
<li>1億3000万ドル超の制作費に対し、興行収入は大きく届かなかったとされ、巨匠の夢と産業の現実が対比されている。</li>
<li>スペインの映画文化は、作家性と産業の衝突を好んで論じる。アルモドバル以降、「映画は社会と作者を語るもの」という読み方が強い。</li>
</ul>
<h3>バルデム舞台デビューとトキスチャ来西</h3>
<ul>
<li>El Pais Cultura は、ハビエル・バルデムが舞台「チェーホフ、チェーホフ！」で本格的に舞台出演すると報じた。</li>
<li>El Mundo Cultura は、ドミニカ出身のラップ・デンボウ歌手トキスチャがスペイン公演を行い、身体性、毒性、うつ、エンパワーメントを語ったと伝えた。</li>
<li>スペイン語圏の文化ニュースは、スペイン国内に閉じない。ラテンアメリカ、米国、欧州をまたぐスペイン語文化圏として流通する。</li>
</ul>
<h2>フラメンコニュース</h2>
<h3>エヘアで「Nuestro Flamenco」公演</h3>
<ul>
<li>エヘア・デ・ロス・カバジェロス市は、リカルド・フェルナンデス・デル・モラルのカンパニーによる「Nuestro Flamenco」を紹介している。</li>
<li>カンテ、ギター、バイレの熟練を近くで味わう公演で、フラメンコを観光ショーではなく総合芸術として見せる内容だ。</li>
<li>地方自治体がこうした公演を担う点に、スペインの地域文化政策が見える。日本の市民ホールで能や民謡公演を支える構図にも近い。</li>
</ul>
<h3>第45回コルドバ・ギターフェスティバル</h3>
<ul>
<li>Teatro Cordoba は、第45回コルドバ・ギターフェスティバルがフラメンコの大物を集め、CNAF70周年を祝うと伝えている。</li>
<li>コルドバはアンダルシアの中心都市の一つで、フラメンコギターの権威と歴史を強く持つ。</li>
<li>ギターはフラメンコの伴奏ではなく、カンテやバイレと対等に魂を語る楽器だ。職人芸への敬意という点では、日本の三味線や尺八にも通じる。</li>
</ul>
<h3>ハエン大学、フラメンコ舞踊入門ワークショップ</h3>
<ul>
<li>ハエン大学は、国際ダンスデーに合わせてフラメンコ舞踊の入門ワークショップを開くと伝えている。</li>
<li>大学がフラメンコを扱うことは、伝統芸能を「古いもの」として保存するだけでなく、教育と身体表現として更新する意味がある。</li>
<li>アンダルシア発祥のフラメンコは、マドリードや北部で全国文化として消費される一方、地元の大学や地方都市では生活に近い学びとして残る。</li>
</ul>
<h2>まとめ</h2>
<p>今日のスペインは、ベネズエラ地震の続報にラテンアメリカへの近さがにじみ、国内では移民正規化と過去の警察利用疑惑が制度を揺らした。文化欄ではファシズム記憶、ソローリャ紛失、映画産業の失敗が並び、フラメンコは地方自治体、コルドバのギター、大学教育の中で生き続けている。スペインは対立を隠さず、地域文化を政治と同じ強さで語る国だ。</p>
