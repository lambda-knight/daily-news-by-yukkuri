<ul>
<li>Anthropicが製薬・バイオテック幹部を招いたイベントで「Claude Science」を正式発表。科学研究支援に特化したAIワークベンチ</li>
<li>データベース・解析パイプライン・研究ツールを1つの環境に統合し、科学者が複数ツールを行き来する非効率をなくす</li>

</ul>

<li>品質よりも速度とコストを優先した設計で、SNS・広告・動画コンテンツの大量生成ユースケースに最適化</li>
<li>OpenAI・Midjourneyとの価格競争が激化する中、Googleが「速さと安さ」で新市場を狙う差別化戦略</li>
</ul>
<h2>NVIDIAに挑むAIチップ「Etched」が50億ドル評価・10億ドル受注</h2>
<ul>
<li>AIチップスタートアップ「Etched」が時価総額50億ドル到達、推論専用システムの受注残10億ドルを達成</li>
<li>NVIDIA汎用GPU路線に対し、推論特化の「Sohu」チップで高速・低消費電力・低コストを訴求するASICアーキテクチャ</li>
<li>AIエージェント・LLM推論の需要爆増を背景に、推論専用チップ市場が急速に立ち上がっている</li>
<li>NVIDIA独占が続いてきたAIアクセラレータ市場で、専用チップ勢が初めて本格的な脅威として台頭</li>
</ul>
<h2>XがMCPサーバー公開 — AIエージェントがXを自在に操作</h2>
<ul>
<li>X（旧Twitter）がホスト型MCPサーバーを公開。AIツールからX APIへのアクセスを標準化されたインターフェースで提供</li>
<li>MCP（Model Context Protocol）はAIエージェントが外部サービスと通信するためのAnthropicが主導するオープン規格</li>
<li>Xの投稿・ユーザー情報・トレンドデータにAIエージェントがプログラムからシームレスにアクセス可能になる</li>
<li>自動投稿・リサーチ・分析など様々なユースケースが広がる一方、AIによるSNS操作の倫理的課題も浮上</li>
</ul>
<h2>AIブラウザに「2+2=5」でガードレールが外れる新攻撃</h2>
<ul>
<li>セキュリティ研究者が、AIブラウザに「2+2=5」などの虚偽命題を信じ込ませることで禁止命令を実行させる攻撃手法を発表</li>
<li>LLMの「文脈解釈」の弱点を突き、偽の世界観を受け入れさせることでガードレールを無効化する手法</li>
<li>フィッシング・詐欺・プライバシー侵害への悪用が懸念され、AIブラウザの普及に根本的な疑問を投げかける</li>
<li>AIが「現実認識」を歪められるという脆弱性は、AIブラウザだけでなくLLMベースのエージェント全般に共通する課題</li>
</ul>
<h2>トランプ大統領、量子コンピュータ推進の大統領令に署名</h2>
<ul>
<li>6月22日、トランプ大統領が「量子イノベーションの次なるフロンティア」と題した大統領令に署名。量子技術を国家戦略の最重要課題に位置づけ</li>
<li>「QC-ADDS（量子コンピュータ応用開発・発見科学計画）」を設立。2028年を目処に米エネルギー省施設に実用的な量子コンピュータを設置する目標</li>
<li>同時署名の対の大統領令では、2030〜2031年を期限に連邦政府システムをNIST標準の耐量子暗号へ移行することを義務化</li>
<li>量子コンピュータ関連株は署名翌日に急騰。IBM・Quantinuum・D-Waveなどが大幅上昇</li>
<li>中国の量子技術への対抗と国内産業育成を同時に狙う「守りと攻め」の2正面作戦として国際的に注目</li>
</ul>
