<h2>33万個のLEGOスポーツカー、時速111kmの世界記録を樹立</h2>
<ul>
<li>LEGOが公式に制作したKoenigseggのスポーツカー型模型車が、なんと実際に走行できるうえ時速111km（69mph）を記録</li>
<li>ボディを構成するLEGOパーツはなんと327,906個、作業時間は数千時間に及ぶ</li>
<li>車体は本物のKoenigsegg「Sadair's Spear」に忠実に再現しつつ、ブロックが崩れないよう特殊な接合技術を採用</li>
<li>走行中もLEGOブロックはしっかり固定されたまま——「ちゃんと速度が上がっても崩れないのか？」という視聴者の心配をあっさり払拭</li>
<li>「世界一速いLEGOカー」として世界記録に公式認定、LEGO公式SNSで1,000万回以上再生</li>
<li>出典: <a href="https://www.odditycentral.com/auto/sports-car-with-body-made-out-of-328000-lego-pieces-sets-new-speed-record.html">Oddity Central</a></li>
</ul>
<h2>「電気代を払って」——中国の街頭に出没するロボット乞食たち</h2>
<ul>
<li>中国の複数の都市で、「電気代を払ってください」「充電させてください」と書いた看板を持ったヒューマノイドロボットが目撃されている</li>
<li>ロボットは本物の乞食そっくりのポーズで路上に座り、通行人に物乞いをする</li>
<li>実はロボット技術の展示やPRのための演出だが、あまりにもリアルで本物と見間違える通行人が続出</li>
<li>「AIに仕事を奪われた人間の代わりにロボットが物乞いをしているのでは」という皮肉なコメントがSNSで大バズり</li>
<li>「これは技術の敗北か勝利か」という哲学的議論まで勃発、中国ネット民を大いに困惑させた</li>
<li>出典: <a href="https://www.odditycentral.com/news/robot-beggars-on-chinas-streets-suggests-no-one-is-dafe-from-the-ai-revolution.html">Oddity Central</a></li>
</ul>
<h2>オーストリアの農家、3年がかりで羊のチェーンメイル（鎖帷子）を完成させる</h2>
<ul>
<li>オーストリアの発明家が、オオカミの噛みつきから羊を守るため、本格的な「鎖帷子（チェーンメイル）」を3年かけて開発・完成</li>
<li>金属製の環を無数に繋いだ本物のチェーンメイルで、羊の首まわりと胴体を覆う設計</li>
<li>「オオカミが噛んでも歯が通らない」という触れ込みで、一応の防御効果も確認済み</li>
<li>ただし羊がこの装備を嫌がり、着せるのに毎回一苦労するという本末転倒な課題が発覚</li>
<li>「中世の騎士の羊バージョンがここに誕生した」とネットで大絶賛、クラウドファンディングの問い合わせも殺到</li>
<li>出典: <a href="https://www.odditycentral.com/animals/austrian-farmer-spends-three-years-creating-chain-mail-to-protect-sheep-from-wolves.html">Oddity Central</a></li>
</ul>
<h2>クロアチアの離島に「家の中の家」——100年以上謎のまま</h2>
<ul>
<li>クロアチアのブラチ島（Brac）に存在する「家の中の家」は、古い石造りの家屋の内部にもう1軒の小さな家が建てられているという不思議な建築物</li>
<li>100年以上前から存在し、地元住民も「なぜ建てられたのか」誰も知らないという謎に包まれた構造</li>
<li>外から見ると普通の石造りの家だが、中に入ると内部に独立した壁・屋根・窓を持つ別の家がそのまま存在する</li>
<li>建築の専門家も「意図的な設計か偶然の改築なのか判断できない」と首をひねるほど珍妙な構造</li>
<li>観光客に「クロアチア版マトリョーシカ建築」と呼ばれ、ひそかな人気スポットになっている</li>
<li>出典: <a href="https://www.odditycentral.com/architecture/the-house-in-a-house-croatias-intriguing-architectural-oddity.html">Oddity Central</a></li>
</ul>
