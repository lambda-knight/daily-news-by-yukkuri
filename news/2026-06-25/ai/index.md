<h1>生成AI最前線 2026年6月25日号</h1>
<h2>エンジニア職はAI時代に最も強い ── SignalFireの新データが示す意外な現実</h2>
<ul>
<li>「AIがエンジニアの仕事を奪う」という言説に反し、SignalFireの最新データではエンジニア職が新規採用に占める割合は増加中</li>
<li>AI導入が進む企業ほど、AIを使いこなせるエンジニアの需要が高まるという逆説的な構造が浮き彫りに</li>
<li>「AIに代替される」より「AIと協働できる人材」への移行が進んでいる</li>
</ul>
<h2>Google AI人材流出が止まらない ── AnthropicへまたもやトップResearcherが移籍</h2>
<ul>
<li>Google DeepMindのResearcher Jonas AdlerとAlexander PritzelがそろってAnthropicに転職すると報道</li>
<li>Noam Shazeer（キャラクターAI創業）やJohn Jumper（AlphaFold開発）らに続く大型人材流出</li>
<li>AI覇権争いの実態は「モデル」ではなく「研究者の争奪戦」であることが改めて鮮明に</li>
</ul>
<h2>OpenAI、独自チップ「Jalapeño」を初公開 ── Broadcom製・推論特化の自社設計</h2>
<ul>
<li>OpenAIがBroadcomと共同設計した初の独自AI推論チップ「Jalapeño（ハラペーニョ）」を発表</li>
<li>NvidiaのGPUに依存しない自社インフラの構築を目指す戦略の一環。推論ワークロードに最適化されている</li>
<li>GoogleのTPU・MetaのMTIA・AmazonのTrainiumに続き、大手テック各社の独自シリコン競争がさらに激化</li>
</ul>
<h2>富士通「PHOTON」── Transformerの最大475倍のスループットを誇る新LLMアーキテクチャ</h2>
<ul>
<li>富士通が独自開発したエルエルエム向けアーキテクチャ「PHOTON（フォトン）」を発表</li>
<li>GPUあたりの処理性能がTransformerの最大475倍に達し、同性能をより少ないGPUで実現できると主張</li>
<li>エルエルエムの運用コスト削減と省エネ化に直結する技術として国内外から注目を集める</li>
</ul>
<h2>企業のAI予算が枯渇寸前 ── 「トークン配給制」の時代へ</h2>
<ul>
<li>社員が細かい業務タスクにAIを多用し、企業のAI利用予算が想定外のペースで消費される問題が顕在化</li>
<li>一部企業は1人あたりの月間トークン上限を設けるなど「トークン配給制」への移行を始めている</li>
<li>AI活用の「量から質へ」の転換が企業に迫られており、ROI（投資対効果）の厳格な管理が急務に</li>
</ul>
<h2>ナデラCEO「学習ループ」戦略 ── 「最強モデル」競争はもう無意味</h2>
<ul>
<li>MicrosoftのナデラCEOが、モデル性能ランキングよりも「データ×AIの学習ループ」こそが競争優位の源泉と主張</li>
<li>企業がAIを使うことで蓄積されるデータをさらにAI改善に活かす「好循環」を構築することが本質</li>
<li>コモディティ化が進むAIモデルの世界で、差別化はデータとフィードバックループの設計にある</li>
</ul>
<h2>全社AIリテラシー研修 ── 導入カギは「安全に使える人材」を先に育てること</h2>
<ul>
<li>生成AIを全社導入する企業が増える中、情報漏洩・誤情報リスクへの対策として従業員向けリテラシー研修が急務</li>
<li>業種・職種を問わず誰もが実践できる生成AIリテラシー教育のフレームワークが注目されている</li>
<li>AIの「使える/使えない」より「安全に使える」を先に確保することが全社導入の鉄則</li>
</ul>
<h2>Facebook AIコンパニオン ── クリエイター向けAIアシスタントアプリを展開</h2>
<ul>
<li>MetaがFacebook上でクリエイター向けに常駐するAIコンパニオンアプリのテスト運用を開始</li>
<li>投稿ネタのアイデア出し・キャプション生成・コメント返信補助など、SNS運用業務を全面的にサポート</li>
<li>コンテンツクリエイターのAI依存が深まる中、メタプラットフォームがクリエイターエコノミーにさらに食い込む</li>
</ul>
<h2>Figma、AIでデザインツールを大刷新 ── コードレイヤー・アニメーション・カスタムプラグインを追加</h2>
<ul>
<li>Figmaが大型アップデートを発表。デザインと開発を直結する「コードレイヤー」機能を新たに導入</li>
<li>アニメーションやシェーダーのサポートも追加。AIでカスタムプラグインを生成する機能も搭載</li>
<li>デザインツールがAI統合で「デザイン＋コード＋動き」の3要素を一元管理できる開発基盤へと進化</li>
</ul>
<h2>Stripe・Anthropic・OpenAIがタッグ ── 呼吸器感染症ゼロを目指す科学投資</h2>
<ul>
<li>StripeのCollison兄弟が主導し、AnthropicとOpenAIが資金提供する呼吸器感染症の予防研究プロジェクトが始動</li>
<li>風邪やインフルエンザをAI創薬・mRNAワクチン技術で「根絶」することを長期ミッションとして掲げる</li>
<li>テック企業がAIを武器にヘルスケア分野へ本格参入する象徴的な動きとして注目を集める</li>
</ul>
<h2>AIのウェブデータ基盤層が台頭 ── 「非構造化データ」をAIに食べさせる新産業</h2>
<ul>
<li>AIモデルの学習・推論に必要なウェブ上の大規模データを提供するインフラ企業の勃興が加速</li>
<li>公開情報の多くはブロックや非構造化状態にあり、AIが利用できる形に変換するデータ基盤ビジネスが急成長</li>
<li>RAGや企業AIの精度向上に直結するデータインフラは、次のAIバリューチェーンの要になりつつある</li>
</ul>
<h2>キオクシア急成長の理由 ── AIブームが生んだNANDフラッシュ需要爆発</h2>
<ul>
<li>キオクシアがAI向けデータセンターのNANDフラッシュ需要急増で業績を大幅に改善</li>
<li>データセンターのストレージ需要はGPU投資の裏で急拡大しており、半導体メモリ市場全体が活況</li>
<li>米国でも同種のメモリ企業が収益4倍増・利益15倍増という驚異的な成長を記録（2025年比）</li>
</ul>
<h2>Agility Robotics、SPAC上場で25億ドル調達へ ── 人型ロボット商業化の新フェーズ</h2>
<ul>
<li>人型ロボット「Digit」を開発するAgility RoboticsがSPAC（特別買収目的会社）経由で上場計画を発表</li>
<li>25億ドル規模の取引で6億2000万ドルの調達を見込む。工場や倉庫向けの商業展開を加速</li>
<li>Figure・1X・テスラ・Boston Dynamicsとの競争が激化する中、SPACによる資金調達でスピードを重視</li>
</ul>
<h2>GMO、国産四足歩行ロボットを陸自駐屯地へ ── 警備省人化の実証が始まる</h2>
<ul>
<li>GMOインターネットグループと未来ロボットが開発した国産四足歩行型警備ロボットが陸上自衛隊駐屯地で実証開始</li>
<li>24時間巡回警備を自動化し、警備要員の省人化を目指す。カメラとセンサーによる異常検知も備える</li>
<li>安全保障・防衛分野への国産ロボット投入は、中国Unitree製品への対抗軸としても位置付けられる</li>
</ul>
<h2>超高精度VRグローブがロボット学習の新ツールに ── Diver-X「ContactGlove3」</h2>
<ul>
<li>Diver-Xが電磁場トラッキング方式の高精度VRグローブ「ContactGlove3」を発表。誤差中央値0.5mmを実現</li>
<li>ロボット産業から注目を集め、ロボットアームの「手先学習」データ収集ツールとしての用途を拡大中</li>
<li>民生用とB2B業務用を展開。ゲームVR向けから産業・ロボティクス向けへとユーザー層が急速に広がる</li>
</ul>
<h2>空飛ぶソーラー通信プラットフォーム ── 成層圏から日本沿岸へインターネットを届ける</h2>
<ul>
<li>米ニューメキシコ州のSceye社が開発した全長約60mの太陽光駆動エアシップが、8月にも日本沿岸上空に展開予定</li>
<li>高度18kmの成層圏に定位し、衛星より低コスト・通信衛星より高帯域な「ヘイルズ（HAPS）」として運用</li>
<li>AIデータセンターの通信インフラとしての活用も視野に入れており、宇宙×AI×通信の融合が加速</li>
</ul>
