<h2>論文 1</h2>
<p><strong>リンドブラディアン・シミュレーションのサンプル計算量を改善</strong>
- arXiv: 2605.30301
- 著者: Siheon Park, Youngjin Seo, Byeongseon Go, Dhrumil Patel, Mark M. Wilde ほか（KAIST・Louisiana State University）</p>
<h3>背景</h3>
<ul>
<li>量子オープン系のシミュレーションには「リンドブラディアン」と呼ばれる方程式が使われる</li>
<li>これを量子コンピュータ上で効率よく実行するには、サンプルをいくつ用意すればよいかが重要課題</li>
<li>既存の波行列線形化（WML）アルゴリズムの計算量界 $O(d^2 t^2/\varepsilon)$ が課題だった</li>
</ul>
<h3>手法</h3>
<ul>
<li>ジャンプ演算子 $L$（次元 $d$）に対して非漸近的な上界を導出
$$n_d^*(t, \varepsilon) \le \frac{2d+3}{8} \|L\|_\infty^2 \cdot \frac{t^2}{\varepsilon}$$</li>
<li>ランダムなリンドブラッド演算子では $\|L\|_\infty^2 = O(1/d)$ となり次元依存性が消える</li>
<li>ランク1演算子での下界 $\Omega(dt^2/\varepsilon)$ も構成し、最悪ケースを特定</li>
</ul>
<h3>結果</h3>
<ul>
<li>典型的なランダム演算子では $O(t^2/\varepsilon)$ と次元に依存しないサンプル数で実行可能</li>
<li>最悪ケースと典型ケースの間に鋭い二分法（シャープ・ダイコトミー）が存在</li>
<li>WML アルゴリズムの理論的基盤を強化</li>
</ul>
<h3>意義</h3>
<ul>
<li>量子シミュレーションにおけるリソース見積もりが大幅に改善</li>
<li>化学・材料科学の開放量子系シミュレーションへの応用が加速する見込み</li>
</ul>

<li>$^{10}$B（ホウ素-10）と $^{12}$C（炭素-12）の核構造を計算対象に</li>

<li>cSD・pnSD マッピングの誤差はそれぞれ 3.37%・8.88%</li>
<li>$^{12}$C では cSD マッピングで 6.82% の誤差</li>
</ul>
<h3>意義</h3>
<ul>
<li>近傍の将来型量子ハードウェアでも核物理計算が可能なことを実証</li>

- arXiv: 2605.30141
- 著者: Masari Watanabe, Hirofumi Nishi, Taichi Kosugi, Shinji Tsuneyuki, Yu-ichiro Matsushita（NTT Research・東京大学）</p>
<h3>背景</h3>
<ul>
<li>量子シミュレーションの基底状態準備は後続アルゴリズム全体のコストに直結</li>
<li>行列積状態（MPS）をキュービットレジスタにロードする「MPS符号化」に深さの壁が存在</li>
<li>密度行列繰り込み群（DMRG）との組み合わせで壁を突破する方法が必要</li>
</ul>
<h3>手法</h3>
<ul>
<li>三段階フレームワーク：DMRG → 最適化不要 MPD 符号化 → 確率的虚時間発展（PITE）</li>
<li>中央ボンドのシュミットランクが層数に対してロジスティック成長することを発見</li>
<li>変曲点 $L^*$ を効率的符号化と非効率領域の境界として特定</li>
<li>DMRG で推定した基底状態エネルギー・有効ギャップ・参照重なりから PITE スケジュールを決定論的に設定</li>
</ul>
<h3>結果</h3>
<ul>
<li>スピン-1/2 磁場交互ハイゼンベルク鎖での数値実験で有効性を検証</li>
<li>$L^*$ で止めることで過度に深い符号化回路を回避</li>
<li>PITE の後選択オーバーヘッドを大幅に抑制</li>
</ul>
<h3>意義</h3>
<ul>
<li>古典 DMRG 前処理・最適化不要 MPS 符号化・決定論的 PITE の三要素を融合</li>
<li>量子シミュレーションの実用的ハイブリッドルートを示す</li>
</ul>
