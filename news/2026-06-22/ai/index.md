<h1>生成AI最前線 2026年6月22日号</h1>
<h2>iOS 27 が変えるスマホ体験 ── Siri の先にある実用AI機能</h2>
<ul>
<li>AppleがWWDCで発表したiOS 27は、Siriの刷新が話題を集めたが、実用的なAI機能は他の部分にも多数搭載</li>
<li>カメラのリアルタイム翻訳・文字起こし強化、写真アプリでの自然言語検索、メールの自動要約と返信候補など</li>
<li>ハードウェア非依存で旧モデルにも一部展開予定。デバイス上処理（オンデバイスAI）で個人情報漏洩リスクを低減</li>
</ul>
<h2>トランプ政権のAnthropicへの圧力 ── 漁夫の利を得るのは誰か</h2>
<ul>
<li>トランプ政権がAnthropicのフェイブル5・ミュトス5の提供停止を命じた余波が続く</li>
<li>AI業界内で恩恵を受けるのはOpenAI・Google・Meta・中国系モデルとの見方が浮上</li>
<li>規制の不公平感から「政治介入はイノベーションを歪める」との批判が研究者・産業界双方から強まっている</li>
</ul>
<h2>AIモデルの中であなたはどう映っているか ── "In the Weights" 登場</h2>
<ul>
<li>元Pinterestエンジニアらが開発した「In the Weights」は、自分がAIモデルにどう認識されているかを可視化するツール</li>
<li>名前・組織・テーマをAIに問い合わせ、複数のエルエルエムの回答を横断的にまとめて表示</li>
<li>グーグル検索の権威が低下しAI検索が台頭する中、「AIモデル上での存在感」が新たなパーソナルブランドの指標に</li>
</ul>
<h2>GMOがUnitree Roboticsの正規代理店に ── 人型ロボット日本市場へ本格参入</h2>
<ul>
<li>GMOインターネットグループ傘下のGMO AI＆ロボティクス商事が、中国Unitree Roboticsと国内正規代理店契約を締結</li>
<li>Unitreeは安価・高性能な人型ロボット「G1」「H1」等で急成長。世界のロボット市場を席巻しつつある中国メーカー</li>
<li>導入相談から保守・サポートまで一気通貫で提供。工場・物流・建設現場向けに国内展開を加速</li>
</ul>
<h2>UK、欠陥を知りながら顔認証で難民審査 ── AI倫理の試練</h2>
<ul>
<li>英国政府が亡命希望者の年齢確認に顔認証AIを導入すると発表。精度に問題があることを自ら認めつつも推進</li>
<li>顔認証による年齢推定は黒人・アジア系の人物に対して誤差が大きく、差別的運用の懸念が専門家から相次ぐ</li>
<li>「欠陥のあるAIで脆弱な人々を審査する」という構造が、AI行政活用における透明性・公正性の議論を加速</li>
</ul>
<h2>Gartnerがプライバシー執行強化を警告 ── 2025年の罰金は5380億円超</h2>
<ul>
<li>ガートナーが発表したレポートによると、2025年に米国の州当局が科したプライバシー法違反の罰金は34億2500万ドル（約5380億円）</li>
<li>これは過去5年間の累計を一年で超えた規模。2028年まで執行強化が続くと予測</li>
<li>エーアイが個人データを大量処理する時代に、CISOはデータガバナンスとAI利用方針の抜本的見直しを迫られている</li>
</ul>
<h2>LLMボトルネック突破の主張に証拠が揃いつつある ── Subquadratic続報</h2>
<ul>
<li>スタートアップ「サブクアドラティック」がアテンション機構の計算量問題を解決したと主張。昨日の続報</li>
<li>当初は詳細非公開で懐疑的な見方が多かったが、実測データを段階的に公開し始め信憑性が増している</li>
<li>二次関数的な計算量を線形以下に抑える手法が実用化されれば、長文処理・推論コスト・リアルタイム応答に革命的変化</li>
</ul>
<h2>BCI臨床試験が急加速 ── 世界各地で同時多発的に始まった脳インプラント研究</h2>
<ul>
<li>ブレイン・コンピュータ・インタフェース（ビーシーアイ）の臨床試験数が過去2年で急増</li>
<li>ALS患者ケイシー・ハレルさんが約3年間にわたり脳インプラントを日常使用し、思考だけでコミュニケーションを実現</li>
<li>エーアイの信号解読アルゴリズムの進化が精度を飛躍的に向上させ、医療・リハビリ・拡張身体の実用化が現実に</li>
</ul>
