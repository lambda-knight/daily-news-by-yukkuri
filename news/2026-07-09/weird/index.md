<h2>38ヶ月で完成した世界最大の駅、重慶に爆誕</h2>
<ul>
<li>中国・重慶の「重慶東駅」が世界最大の駅として認識される。面積は122万平方メートルで、ニューヨークのグランドセントラル駅の6倍、ドイツ・ライプツィヒ中央駅の15倍</li>
<li>プラットフォーム29基、線路15本。建物は最大8階建てで、屋根に使われた鋼材だけで16,500トン。ピーク時は1時間に16,000人の乗客を処理できる能力を持つ</li>
<li>建設期間はわずか38ヶ月。最大4万人の作業員を動員し、コンクリート約200万立方メートル、鋼材36万6000トンを投入。山岳地形のため山を切り崩して用地を確保した</li>
<li>出典: <a href="https://www.odditycentral.com/architecture/china-builds-worlds-largest-train-station-in-just-38-months-using-an-army-of-workers.html">Oddity Central</a></li>
</ul>
<h2>「サボり社員をクビにしたい」英企業、監視したら全員がサボっていた</h2>
<ul>
<li>イギリスのある企業、労働法が厳しく成績の悪い社員を簡単に解雇できず、経営陣が一計を案じる</li>
<li>「勤務中の私用インターネット利用」を理由に狙いを定めた社員を解雇しようとするも、差別だと訴えられないよう周囲の社員11人まとめてネット利用状況を監視することに</li>
<li>ふたを開けてみると、標的にしていた社員こそが11人の中で最もネット利用が少ないことが判明。他の10人の方がはるかにサボっていた</li>
<li>部署ごとクビにするわけにもいかず作戦は白紙撤回。結局その社員は「勤務時間中に個人事業を運営していた」という別の理由で、後日ようやく解雇された</li>
<li>出典: <a href="https://twistedsifter.com/2026/07/the-surveillance-trap-why-a-tech-savvy-management-team-is-reeling-after-accidentally-proving-that-every-employee-is-wasting-time/">Twisted Sifter</a></li>
</ul>
<h2>父の日、何度聞いても答えなかった夫「やっぱりビーチに行きたい」</h2>
<ul>
<li>アメリカのある女性、父の日に何をしたいか夫に何度も確認</li>
<li>夫が「ビリア（メキシコの煮込み料理）を作ってほしい」と答えたため、食材・花・風船・カードまで用意して準備万端に</li>
<li>ところが当日の朝、料理を仕込んでいる最中に夫が突然「やっぱりビーチに行きたい」と言い出す</li>
<li>妻は「ビーチの準備なんて全くしていないのに」と困惑。手伝いを求めたが夫は「もう何もしたくない」と拒否し、結局ビーチには行かず夫は終日不機嫌に</li>
<li>ネット上では妻に同情する声が大多数だった</li>

</ul>
<h2>コロナ禍の流産で不当解雇→呼び戻されて最小限労働→産休直前に円満退職</h2>
<ul>
<li>アメリカの大手小売店で働く女性、コロナ禍に流産を経験。医師の診断書があったにもかかわらず欠勤を理由に不当解雇される</li>
<li>人手不足に陥った会社が数年後に彼女を再雇用</li>
<li>復帰後は意図的に必要最小限の力だけで勤務。やがて妊娠を報告し、医学的な制限を理由に業務量も抑えてもらう</li>
<li>12週間の有給産休を取得し、復帰予定日の前日に退職。「かつての優秀な働き者」が戻ってくると期待していた経営陣の目論見は外れ、産休中の給料もまるごと無駄になった</li>
