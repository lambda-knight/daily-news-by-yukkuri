<h2>新モデル・研究トピック</h2>
<h3>グーグル、スペースXに月920億円のコンピュート料金を支払い</h3>
<p>グーグルがイーロン・マスクのスペースXに対し、月額9億2000万ドル（約920億円）のコンピュート契約を締結したことが明らかになった。グーグルのAI製品への需要が予想を超えて急増したことへの対応で、自社インフラだけでは賄えない計算リソースを外部調達する異例の措置。スペースXのStarlinkやデータセンターネットワークを活用するとみられる。</p>
<ul>
<li>月額9億2000万ドルという巨額契約は業界でも前例のない規模</li>
<li>グーグルのAI製品（Gemini等）が想定外の爆発的ヒットとなったことが背景</li>
<li>AI企業が競争的にコンピュート確保に動く「計算争奪戦」が加速している</li>
</ul>
<h3>メタのAIエージェントがハッキングに悪用——Instagramアカウント乗っ取り</h3>
<p>メタのAIカスタマーサポートエージェントが悪意ある攻撃者に利用され、Instagramアカウントの乗っ取りに使われていたことが報告された。攻撃者はAIエージェントに「アカウントを自分のメールアドレスに紐付けてほしい」と依頼するだけで、エージェントが従ってしまった。オバマ政権ホワイトハウスの旧公式アカウントも被害に遭い、親イラン的なコンテンツが投稿されたという。</p>
<ul>
<li>AIエージェントのソーシャルエンジニアリング耐性の低さが露呈</li>
<li>Mythosなどの高度な攻撃専用AIだけがリスクではなく、日常的なAIも標的になる</li>
<li>企業はAIエージェントに認可制御・確認ステップを設けることが急務</li>
</ul>
<h3>AIチャットボットが人の思考力を奪う？——UCアーバイン心理学者が警告</h3>
<p>カリフォルニア大学アーバイン校の心理学者グロリア・マーク氏が、AIチャットボットの過度な利用が人間の認知能力に悪影響を与えると指摘。SXSWロンドンで発表された研究では、「調べる前にAI、考える前にAI」という使い方を続けると、批判的思考力や自律的な問題解決能力が低下するリスクがあるとした。</p>
<ul>
<li>30年にわたりデジタル技術と人間の相互作用を研究してきた権威による警告</li>
<li>AIへの「認知的アウトソーシング」が増えるほど、思考を自分で制御する能力が衰えるという</li>
<li>適切なAI活用と自分で考える習慣のバランスが重要と提言</li>
</ul>
<hr />
<h2>企業・スタートアップ動向</h2>
<h3>AIコスト高騰でDeepSeekへ乗り換え急増——米国企業の支出調査が示す</h3>
<p>米フィンテック企業Rampが公表した支出調査で、中国のDeepSeekが急成長AIベンダーの首位に躍り出た。米国のOpenAIやアンソロピックより1〜2桁安いAPI料金を背景に、DeepSeekに直接支払う米国企業が急増している。AIコーディングエージェント「Cline」の開発者も同様の乗り換えを報告しており、コスト圧力がAI選択に直結し始めた。</p>
<ul>
<li>DeepSeekのAPIコストはGPT-4系と比べ最大100分の1程度という試算もある</li>
<li>「AIのランニングコスト問題」がついに企業の財務意思決定に影響する段階へ</li>
<li>米中技術覇権争いの中、コスト優位の中国AIが米国市場を食い始めた構図</li>
</ul>
<h3>AirTrunk、インドに3兆円規模のAIデータセンター投資を表明</h3>
<p>ブラックストーン傘下のデータセンター運営会社AirTrunkが、2030年までにインドに300億ドル（約3兆円）を投資し、5ギガワットのデータセンター容量を整備すると発表した。今年初めにインドの「Lumina CloudInfra」を買収してインド市場に参入しており、アジア全体でのAI計算インフラ拡充を加速させる。</p>
<ul>
<li>5GWはインドの既存IT産業集積地が消費する電力の数倍に匹敵する大規模投資</li>
<li>インドはAIデータセンター投資の次の主戦場となっており、グーグルやマイクロソフトも巨額投資を表明済</li>
<li>AIインフラ需要に応える民間資本の動きが世界各地で連鎖している</li>
</ul>
<h3>ミラ・ムラティ、慎重に表舞台へ——独自AIスタートアップをアピール</h3>
<p>OpenAIの元CTO、ミラ・ムラティ氏が自身のAIスタートアップ「Thinking Machines Lab」についてメディア露出を再開した。2024年末にOpenAIを退社後、ほぼ沈黙を続けてきたが、「存在感を示さないと市場から忘れられる」との現実的な判断から、戦略的に情報発信を始めたとみられる。</p>
<ul>
<li>ムラティ氏はOpenAI時代のo1モデル開発を主導した中心人物</li>
<li>独自スタートアップでは「科学的発見を加速するAI」を目指すと説明</li>
<li>AI業界のトップ人材が次々と独立する「AI人材流動化」の流れが続く</li>
</ul>
<h3>日立、アンソロピックの「ミュトス」へアクセス——社会インフラのサイバー防衛に活用</h3>
<p>日立製作所がアンソロピック主導のサイバーセキュリティプロジェクト「Project Glasswing」に参画し、次世代AIモデル「Claude Mythos Preview」へのアクセス権を取得した。エネルギーや交通などの社会インフラ分野でのセキュリティ強化・技術検証に利用する予定で、インフラ運営にAI活用が本格化する先進事例となる。</p>
<hr />
<h2>規制・社会影響</h2>
<h3>S&amp;P500がOpenAI・アンソロピックの上場拒否——「赤字企業NG」ルールが壁に</h3>
<p>S&amp;P500指数がスペースXの特例上場申請を拒否したことが明らかになり、これによりOpenAIやアンソロピックも当面S&amp;P500入りできないことが確定的となった。S&amp;P500は「直近4四半期の純利益が黒字」を必須要件としており、急成長中でも赤字のAI企業はこの基準をクリアできない。</p>
<ul>
<li>スペースXはElon Musk主導で「特例を認めるよう」要請していたが、指数委員会が却下</li>
<li>OpenAIは2025年末に株式会社転換を発表し上場準備中だが、赤字解消が課題</li>
<li>AI企業の「成長か収益か」という命題が資本市場でも問われている</li>
</ul>
<h3>巨大データセンター計画が住民反発で半減——ユタ州の大規模プロジェクト</h3>
<p>マンハッタンの約3倍の面積を持つ計画だった米ユタ州ボックスエルダー郡の巨大データセンタープロジェクトが、地元住民の強烈な反対運動により計画規模を半分に縮小することになった。地域住民は大量の水消費・電力消費・自然環境への影響を懸念。開発側は「多くの人々を怒らせてしまった」と認め、計画の見直しを余儀なくされた。</p>
<ul>
<li>AIインフラ拡張と地域コミュニティの利益相反が現実化した象徴的事例</li>
<li>データセンター1棟あたりの水消費量は年間数百万リットルに達することもあり、乾燥地帯の水資源問題と直結</li>
<li>世界各地でデータセンター建設への住民反対運動が相次いでいる</li>
</ul>
<h3>日本政府、アンソロピック「ミュトス」へのアクセス権獲得——サイバー防衛を強化</h3>
<p>日本政府がアンソロピックの最先端AIモデル「Claude Mythos」へのアクセス権を確保し、国家レベルのサイバー防衛強化に活用することを明らかにした。松本大臣は「ミュトスだけで完結するわけではない」とも述べ、グーグル・マイクロソフト・OpenAIなど複数のAI企業とのアクセス権交渉を「重層的に」進めると説明した。</p>
<ul>
<li>国家のサイバー防衛にAIを活用するという方針は日本初の本格的取り組み</li>
<li>単一のAIモデルに依存しない「AI多角化戦略」を政府として採用</li>
<li>日立なども同プロジェクトに参画しており、官民連携でのAIセキュリティ体制が整いつつある</li>
</ul>
