## 概要

今日のスペインニュースをずんだもんと四国めたんがお届け！国際面は米国建国250周年を迎えたトランプ大統領のラシュモア山演説、ローマ教皇レオ14世によるランペドゥーサ島でのEU移民政策批判、イラン最高指導者ハメネイ師の葬儀をめぐる反米デモ、「真っ二つに割れるラテンアメリカ」の論説と、地中海・ラテンアメリカ・米国をめぐるスペインらしい視点をお伝えします。国内はアンダルシア州PP・Vox連立合意の具体的な中身（減税1,300万ユーロ・教育助成拡大）、国税庁内部の混乱、ETAテロリスト「モンドラゴンの肉屋」死去に伴う記憶の再検証、マドリードのプライド2026パレードと、右派連立・地方分権・歴史の記憶・社会運動が同時に動く一日でした。文化面はスペイン国外で最も聴かれる音楽家Rels B、サン・フェルミン祭の準備、ハビエル・バルデム、ガエル・ガルシア・ベルナルのインタビューが話題に。そしてフラメンココーナーでは、ラテン・グラミー2度受賞のギタリスト、アントニオ・レイ、ベテランカンタオールのエル・ペレ、新作を発表したカンタオーラのサンドラ・カラスコ、名門ファルーコ一族のバイラオール、ファルキート、さらにカマロン・デ・ラ・イスラ生誕75周年の大型追悼コンサート情報まで、大物・注目株のニュースをまとめて紹介します。

<h2>国際情勢（スペイン視点）</h2>
<ul>
<li>米国が独立250周年を迎えたが、トランプ大統領がラシュモア山で「自分の顔が刻まれることを望んでいる」とも言われる演説を行い、冷戦期さながらに共産主義の亡霊を攻撃する超大げさな言葉を並べた。首都ワシントンは記録的な熱波にも見舞われ、「トランプが自らを独立記念日の主役に仕立てた、分断された国家の250年」という厳しい論調で報じられている</li>
<li>ローマ教皇レオ14世が、地中海への玄関口ランペドゥーサ島で欧州連合の移民政策を批判。「この海で亡くなった人々は、下された、あるいは下されなかった決定の犠牲者だ」と述べ、「力の論理」か「無関心という名の力の論理」かを選ぶよう欧州に迫った</li>
<li>イラン最高指導者ハメネイ師（イスラエルと米国により殺害）の葬儀をめぐり、後継者と目される息子モジュタバ氏が参列するかどうかに注目が集まっている。同時に葬儀はテヘランで数万人を動員する反米デモの場と化し、「復讐だ、アメリカに死を」の声が響いた。米国建国250周年という米国側の祝賀ムードと真っ向から対立する構図が report された</li>
<li>「真っ二つに割れるラテンアメリカ」と題する論説。過激な政治提案が、大陸のあらゆる社会階層で支持を広げていると分析している</li>
</ul>
<h2>スペイン国内ニュース</h2>
<ul>
<li>アンダルシア州の与党・国民党（PP）と極右Voxの連立合意の具体的な中身が判明。1,300万ユーロ規模の減税、高校（バチジェラート）への公的助成拡大、社会団体向け補助金の削減などが柱。モレノ州首相は以前「Voxとの政権は不可能」と言い切っていたが、わずか議席2票差のためにVoxの主張をほぼ丸のみした形になった</li>
<li>（比較）連立のために少数会派の主張を大きく受け入れる姿は、日本の連立政権交渉にも通じるが、スペインでは「かつて不可能と断言したこと」を丸のみする様子がここまであけすけに報じられるのが特徴的</li>
<li>国税庁（アヘンシア・トリブタリア）内部で不満が噴出。レジスラチュラ（今国会期）終盤、カタルーニャ民族主義政党との合意事項の履行が焦点となる中、長官アルカディ・エスパーニャ氏について「意欲満々で着任したのに、上から権限を奪われている」との内部の声が報じられた</li>
<li>（比較）中央省庁の人事・権限問題が、地域政党との政治的取引によって直接左右される構図は、地方分権をめぐる対立が日常的に行政の内部にまで及ぶスペインらしさを示している。日本では省庁人事がここまで特定地域政党の意向に紐づくことは少ない</li>
<li>「モンドラゴンの肉屋」と呼ばれたETAテロリスト、ヘスス・マリ・サバルテ受刑者の死去を機に、かつて行われた戦慄のインタビューが改めて注目されている。子どもにさえ心を動かされなかったとされる人物が「お前を見張っていた、お前はチャクラ（警官の蔑称）だと思っていた」と語った内容が振り返られ、バスク地方のテロの記憶が生々しく蘇った</li>
<li>（比較）テロリストへの直接インタビュー記録が、本人の死後もメディアで詳細に再検証される文化は、日本におけるオウム真理教や日本赤軍関係者の扱いとは温度差がある。スペインは加害者本人の言葉を長く保存し、繰り返し社会に問い直す傾向が強い</li>
<li>マドリードで「オルグージョ（プライド）2026」パレードが開催され、気温38度の炎天下にもかかわらず数千人がLGBTI+の権利を訴えて行進。参加者の一人は「スペインは私に扉を開いてくれた…今日、私は自由な男だと感じる」と語った</li>
</ul>
<h2>文化・芸能ニュース</h2>
<ul>
<li>スペイン国外で最も聴かれているスペイン人ミュージシャン、Rels Bのインタビュー。18歳のとき左官業をしながらストリートアーティストをプロデュースしていた過去を振り返り、「自分は仕事をしているんじゃない、疲れる“趣味”を持っているだけだ」と語った。現在Spotifyで毎月2,400万人が彼の都会的なサウンドを聴いているという</li>
<li>パンプローナの「サン・フェルミン祭」の準備が本格化。牛追い（エンシエロ）や闘牛など516の催しが予定され、投資額は176万ユーロ。世界的に有名な祭りだが、その経済効果の全容は実は把握しきれていないと報じられている</li>
<li>俳優ハビエル・バルデムのインタビュー。「日常のルーティンに流されそうになる時もあるが、そんな時はペネロペ（・クルス）の目を見て『なんて美しいんだ』と思う」と語った。監督ロドリゴ・ソロゴエンによる新作『El ser querido』が夏の終わりに公開予定で、衝撃作という評判のテレビドラマ版リメイクにも新境地で挑む</li>
<li>メキシコ出身の俳優ガエル・ガルシア・ベルナルのインタビュー。フィリピン人監督ラヴ・ディアスの新作でマゼランを演じ、「こんなに滑稽で保守的な時代だからこそ、思い切り実験的で“イカれた”ことをやるべき時だ」と植民地主義の爪痕や映画の力について語った</li>
</ul>
<h2>フラメンコニュース</h2>
<ul>
<li><strong>ギタリスト</strong>: ラテン・グラミー賞を2度受賞したアントニオ・レイが、新作アルバム『フラメンコ・シン・フロンテーラス（国境なきフラメンコ）』を携えて活動中。9月開催のセビリア・フラメンコ・ビエナルでこのアルバムを披露する予定で、現在も精力的にツアーを続けている</li>
<li><strong>カンタオール</strong>: 伝説的なベテランカンタオール、エル・ペレが、7月19日から26日にムルシア州で開催される「ロ・フェロ音楽祭」に出演。ドランテス、パコ・セペロ、サンドラ・カラスコら豪華な顔ぶれが揃うフラメンコ界屈指のラインナップとなっている</li>
<li><strong>カンタオーラ</strong>: サンドラ・カラスコが新作アルバム『ヌンカ・メ・オルビデス（決して私を忘れないで）』を発表。母と娘の絆に捧げた作品で、ギタリストのダビ・デ・アラハルと共に、記憶とアンダルシアのルーツから生まれたプロジェクトだという。8月1日にはマドリードの夏フェスティバル「ベラーノス・デ・ラ・ビジャ」でも披露される</li>
<li><strong>バイレ</strong>: 名門ファルーコ一族の看板、バイラオール、ファルキートが7月12日、マドリードのコンデドゥケ中庭で伝統舞踊のリサイタルを行う。ヒターノ（ロマ）の家系に根差した舞台で、セギリージャ・タンゴス・アレグリアス・ソレアといった多彩な演目を、飾らない“舞台の真実”として披露する</li>
<li>別の大きな動きとして、伝説の歌い手カマロン・デ・ラ・イスラの生誕75周年を祝う「アニョ・カマロン（カマロンの年）」の集大成として、12月1日にマドリードのモビスター・アリーナで大規模な追悼コンサートが開催されることが判明。トマティート、ホセ・メルセ、カルメン・リナーレス、イスラエル・フェルナンデスら世代を超えた豪華アーティストが参加を表明しており、フラメンコ界全体を巻き込む一大イベントになりそうだ</li>
</ul>
<h2>まとめ</h2>
<ul>
<li>国際面は米国建国250周年とトランプ大統領の演説が焦点。ラシュモア山での大げさな演説、記録的熱波、そして「分断された国家」という総括が印象的だった。ローマ教皇の地中海移民政策批判、イラン最高指導者の葬儀をめぐる反米デモも、米国の祝賀ムードと対照的な構図として報じられた</li>
<li>スペイン国内はアンダルシアでのPP・Vox連立の具体的な中身、国税庁内部の混乱、ETAテロリストの死去に伴う記憶の再検証、マドリードのプライドパレードと、右派連立・地方分権・歴史の記憶・社会運動という4つの軸が同時に動いた</li>
<li>文化面はRels Bの音楽的成功、サン・フェルミン祭の準備、ハビエル・バルデムとガエル・ガルシア・ベルナルという2人のスター俳優のインタビューが目立った。フラメンコはギタリストのアントニオ・レイ、ベテランカンタオールのエル・ペレ、新作を出したカンタオーラのサンドラ・カラスコ、名門バイラオールのファルキートと、各分野の大物が勢ぞろい。さらにカマロン・デ・ラ・イスラ75周年の大型追悼コンサートも判明し、フラメンコ界全体が沸き立つ1日となった</li>
</ul>
