<h2>Anthropicがサムスンと独自AIチップを協議</h2>
<ul>
<li>Anthropicがサムスン電子と、自社専用のAIチップ開発について協議を進めていると報じられた</li>
<li>OpenAIが約1週間前にブロードコムとの提携で自社製AIチップを発表した直後の動き</li>
<li>大手AI企業がNVIDIA依存から脱却し、独自半導体を持とうとする流れが加速している</li>
<li>実現すればAnthropicにとって計算基盤のコスト削減と供給安定化につながる可能性がある</li>
</ul>
<h2>Microsoftが25億ドル規模の自社AI導入会社を設立</h2>
<ul>
<li>Microsoftが顧客企業へのAI導入を専門に手がける新会社を立ち上げ、25億ドルの投資を表明した</li>
<li>Amazon、OpenAI、Anthropicに続き、大手が相次いで「AI導入専業」組織を設けている</li>
<li>企業向けにAIモデルを実際の業務に組み込むまでの支援を専門チームが担う</li>
<li>モデル開発競争に加え、導入・定着支援の争いが新たな戦場になりつつある</li>
</ul>
<h2>OpenAIが米政府に株式5%を提供する案を提示</h2>
<ul>
<li>OpenAIのサム・アルトマンCEOが、米政府向けに自社株式の5%を拠出する案を持ちかけていると報じられた</li>
<li>3月の資金調達時点の企業価値852億ドルで計算すると、5%はおよそ426億ドル相当</li>
<li>構想はアラスカ州の石油収入基金「アラスカ永久基金」のような仕組みをモデルにしている</li>
<li>OpenAIだけでなくAnthropic、Google、Metaなど主要AI企業にも同様の拠出を求める案とされる</li>
<li>実現には議会の関与が必要とみられ、各社が応じるかは不透明な段階</li>
</ul>
<h2>軍事AI規制法案とAI兵器倫理を巡る攻防が続く</h2>
<ul>
<li>米上院で6月、軍のAI利用を制限する新法案が提出され、自律型兵器や核兵器へのAI適用に規制をかける内容が盛り込まれた</li>
<li>国連も6月、AIの軍事利用について各国が議論する非公式協議をジュネーブで開催</li>
<li>フランスのAI企業ミストラルは、教皇レオ14世が回勅でAI兵器への厳格な倫理制約を求めたことに対し、「対立国もAIを使っている以上、欧州も独自の手段を持つ必要がある」と反論</li>
<li>軍事とAIを巡る規制強化派と実用推進派の綱引きが、政治・宗教・産業を巻き込んで広がっている</li>
</ul>
<h2>UCFが量子コンピュータの新しい誤り耐性手法を発表</h2>
<ul>
<li>米セントラルフロリダ大学（UCF）の研究チームが、量子計算の誤りに強い新手法を開発したと発表</li>
<li>微小な機械振動と超伝導系を組み合わせ、量子状態を「編み込む」ように安定化させるトポロジカルな手法を採用</li>
<li>ひも状のパターン全体に着目することで、局所的なノイズの影響を受けにくくする狙い</li>
<li>医療・エネルギー・新素材開発など、将来の実用的な量子計算応用に向けた基盤技術として期待されている</li>
</ul>
<h2>Meta、AIミニゲーム作成アプリ「Pocket」を静かに公開</h2>
<ul>
<li>Metaが実験的なAIアプリ「Pocket」を、大きな告知なしに公開した</li>
<li>テキストで指示するだけでインタラクティブなミニゲームを生成し、共有できる仕組み</li>
<li>生成AIでコードを書かせる「バイブコーディング」の手法をゲーム作りに応用した形</li>
<li>SNS上での二次拡散を意識した、カジュアルな遊び方を狙った設計とみられる</li>
</ul>
<h2>インドの起業家がAI版オフィスソフト「Neo」に3000万ドルを投資</h2>
<ul>
<li>インドのテック起業家バービン・トラキア氏が、私財3000万ドルを投じてAI搭載の業務ソフト「Neo」を開発</li>
<li>同氏にとって5つ目のベンチャーで、企業向けソフトウェア分野への再挑戦となる</li>
<li>Microsoft OfficeやGoogle Workspaceに、AIを前面に出した機能で対抗する狙い</li>
<li>巨大プラットフォームが握る業務ソフト市場に、AIを武器にした挑戦者が現れた形</li>
</ul>
<h2>SpaceXがAIデバイスの試作品を投資家に披露</h2>
<ul>
<li>SpaceXが非公開の投資家向け説明で、「携帯電話のような」AIデバイスの試作品を見せていたと報じられた</li>
<li>詳細は不明だが、通話やメッセージなど携帯電話に近い機能を想定しているとみられる</li>
<li>同社は衛星通信サービス「スターリンク」を展開しており、無線事業への本格参入を狙う兆候との見方も</li>
<li>実現すれば、AIハードウェア市場に宇宙・通信企業という異色の新規参入者が加わることになる</li>
</ul>
<h2>Jersey Mike'sのIPO資料に見るAIブームの過熱ぶり</h2>
<ul>
<li>サンドイッチチェーン「Jersey Mike's」の新規株式公開（IPO）の開示資料に、AIに関する記述が盛り込まれていた</li>
<li>一見AIと無関係なサンドイッチ店の資料にまでAI言及が及んでいることが話題に</li>
<li>投資家向け資料でAIへの言及を入れることが一種の「定型文化」になっている実態を象徴する事例として紹介された</li>
<li>AIブームの過熱ぶりを測る、やや皮肉めいた指標として取り上げられている</li>
</ul>
