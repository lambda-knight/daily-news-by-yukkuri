<ul>
<li>AIインフェレンス（推論）スタートアップのBasetenが、数ヶ月前の大型調達に続き新たに15億ドルを調達中と報じられた</li>
<li>評価額は130億ドルに達する見込みで、モデル推論インフラへの需要爆発を反映</li>
<li>学習より「実際に使う」推論フェーズへの投資が急増し、インフェレンスが次のホットスポットに</li>
</ul>
<h2>OpenAI、IPO前に大物を相次ぎ招聘 ── トランスフォーマー共同発明者も加入</h2>
<ul>
<li>OpenAIがIPO（株式公開）直前に大物人材を次々と確保</li>
<li>Google DeepMindからトランスフォーマー共同発明者のノアム・シャジール氏が移籍</li>
<li>元トランプ政権のAI政策担当官ディーン・ボール氏も同週に加入し、政官双方への影響力を強化</li>
</ul>
<h2>AmazonがNvidiaに正面挑戦 ── 自社AIチップを外部データセンターに販売</h2>
<ul>
<li>AWSが自社開発のAIチップ（Trainium/Inferentia）を外部のデータセンターに販売する交渉を開始</li>
<li>CEOアンディ・ジャシー氏は「500億ドル規模のビジネス機会」と位置づけ</li>
<li>Nvidiaへの依存を減らしたい企業需要を取り込み、半導体市場の競争が激化</li>
</ul>
<h2>AIデータセンターへの電力接続を政府が優遇 ── FERCが「ファストレーン」指定</h2>
<ul>
<li>米連邦エネルギー規制委員会（FERC）が電力系統オペレーターに対し、データセンターへの系統接続を優先するよう指示</li>
<li>大量の電力を必要とするAIデータセンターが送電網への接続を通常より速く確保できる仕組みを整備</li>
<li>一方で電力供給不足自体の解決策は示されておらず、エネルギー問題の根本解決は棚上げ状態</li>
</ul>
<h2>Snapがアリービデオチームを分社化 ── 「Dotmo」設立、コスト圧縮が狙い</h2>
<ul>
<li>SnapchatのAI動画開発チームが「Dotmo」として独立した新会社に分社化</li>
<li>SNS本体のコスト削減を進める一方、急成長するAI動画市場に特化した組織を設立</li>
<li>AI動画はSoraやRunway、Kling AIなどが競争する注目市場で、大手SNSも切り離して勝負</li>
</ul>
<h2>バーニー・サンダース議員、AI産業を市民の手に ── 7兆ドル超の大胆計画を発表</h2>
<ul>
<li>米民主社会主義者のサンダース上院議員がAI大手から数兆ドルを移転させる法案を発表</li>
<li>計画総額は7兆ドルに達し、AI産業の公有化・市民管理を目指す内容</li>
<li>AI大手には衝撃的な内容だが、AIの利益が一部企業に集中することへの社会的反発を象徴</li>
</ul>
<h2>Claude Mythos登場でセキュリティが変わる ── AIエージェント時代の防衛策</h2>
<ul>
<li>新AIモデル「Claude Mythos」の登場で脆弱性発見能力が大幅に向上し、攻撃の現実化が月単位から時間単位に</li>
<li>企業ではAI利用のルール・管理体制が追い付いておらず、防衛側の対応が急務</li>
<li>AIエージェントが攻撃ツールとして悪用されるリスクに対し、AIを活用した防衛も必要に</li>
</ul>
<h2>「牧歌的なAI開発の時代が終わった」 ── チームみらい・安野氏がミュトス停止騒動に警鐘</h2>
<ul>
<li>チームみらいの安野貴博氏が、AnthropicのMythos 5・Fable 5提供停止騒動を受けてコメント</li>
<li>「AIを気軽に開発・デプロイできた時代は終わった」と指摘し、AI安全性を巡る転換点を強調</li>
<li>高性能モデルの登場が社会・政治的リスクを高める新フェーズへの移行を示唆</li>
</ul>
<h2>「シャドーAI」7割超の企業が対策追い付かず ── ガートナー調査</h2>
<ul>
<li>調査会社ガートナーが日本企業へのシャドーAI実態調査を発表</li>
<li>73%の企業が、従業員が会社非公認のAIツールを業務利用することへの対策ができていないと回答</li>
<li>「会社が選んだAIだけ使う」ポリシーの形骸化が深刻で、データ漏洩・コンプライアンスリスクが拡大</li>
</ul>
<h2>NRI流AIモデル選定術 ── 「ベンチマークだけで優劣は決まらない」</h2>
<ul>
<li>野村総合研究所（NRI）のAI導入支援担当者が、実務でのAIモデル選定の考え方を解説</li>
<li>コーディングや自律タスクのベンチマークは参考に留め、実際の業務フローでの動作検証が重要と指摘</li>
<li>企業固有のデータ・業務プロセスに合ったモデル選択が成否を左右する</li>
</ul>
<h2>AIコーディングが5年で急進化 ── NTT tsuzumi 2開発者が分析</h2>
<ul>
<li>NTT人間情報研究所の開発者が「Interop Tokyo 2026」でAIコーディング急進化の背景を解説</li>
<li>2021年頃に登場したコーディングLLMが、わずか5年で競技プログラミングレベルの問題を解けるまでに成長</li>
<li>コード生成の学習データが他分野より品質が高く評価しやすいことが急進化の主要因</li>
</ul>
<h2>AIとエッセイの創造性 ── 使う学生・使わない学生、どちらが創造的？</h2>
<ul>
<li>ジョージタウン大学の研究者が、AIを使う学生とAIなしの学生のエッセイを比較する実証実験を実施</li>
<li>AI使用グループのエッセイは均質化傾向が見られ、多様性・独自性が低下するという結果に</li>
<li>「LLMの均一化効果」が創造的多様性にどう影響するかを定量的に示した初期の研究成果</li>
</ul>
