<h2>お湯ゼロで作るエスプレッソ——超音波コーヒーが登場</h2>
<ul>
<li>研究者が超音波振動だけでエスプレッソと同等のコーヒーを抽出することに成功</li>
<li>熱湯を一切使わない——冷水＋超音波の衝撃波で豆の成分を溶け出させる</li>
<li>従来のエスプレッソマシンに比べてエネルギー消費が約75パーセント削減</li>
<li>味は盲目テストで通常エスプレッソと「区別がつかない」と評価</li>
<li>夏場にアイスコーヒーを直接作れる可能性もあり</li>
<li>出典: <a href="https://www.wired.com/story/scientists-brew-espresso-with-ultrasonic-waves/">Wired</a></li>
</ul>
<h2>新しいSiriは本当に賢くなった——ハンズオンレポート</h2>
<ul>
<li>アップルが刷新したSiri AIは「会話型・どこでも使える・本当に役に立つ」と評される</li>
<li>以前のSiriは「わかりません」「検索しますか？」の繰り返しで有名だった</li>
<li>新Siriはアプリをまたいだタスクをこなし、文脈を理解して回答する</li>
<li>「何年も揶揄されてきたSiriが、ついに冗談のネタにならない存在になった」</li>
<li>ただし「賢くなった分、怖い」というコメントも多数</li>
<li>出典: <a href="https://www.wired.com/story/siri-ai-hands-on-iphone/">Wired</a></li>
</ul>
<h2>頑張りすぎたインターンを「クビにしろ」とマネージャーが激怒</h2>
<ul>
<li>あるインターンが毎日定時より2時間早く出勤し、誰よりも仕事をこなした</li>
<li>するとマネージャーが激怒——「このインターンは社員を追い出そうとしている！」と主張</li>
<li>実際には単純に「仕事が好きで早起きが得意だった」だけというオチ</li>
<li>SNSで話題となり「これが昇進を妨げる上司の典型」「インターン可哀想すぎる」と炎上</li>
<li>世界中の読者が「あるある話」として共感中</li>
<li>出典: <a href="https://twistedsifter.com/2026/06/interns-incredible-work-ethic-backfires-after-manager-accuses-them-of-trying-to-oust-the-staff/">Twisted Sifter</a></li>
</ul>
<h2>15年間棚に眠ってたエボラワクチンが急遽出動</h2>
<ul>
<li>2011年に開発されたエボラウイルスワクチンが、これまで一度も使われず研究室で眠っていた</li>
<li>コンゴ民主共和国で発生中のブンジブギョ型エボラウイルスの流行を受けて急遽テスト開始</li>
<li>理由は「今まで流行したエボラのタイプが違った」から——専用ワクチンが使えなかった</li>
<li>「存在するのに使えない」という皮肉な状況が15年も続いていた</li>
<li>研究者たちは「ようやくワクチンの出番が来た」と複雑な心境</li>
<li>出典: <a href="https://www.wired.com/story/ebola-vaccine-sitting-on-shelf-for-15-years/">Wired</a></li>
</ul>
