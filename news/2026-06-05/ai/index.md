<h2>新モデル・研究トピック</h2>
<h3>エストニア政府ベンチマーク──ロシアのプロパガンダに最も強いエルエルエムは？</h3>
<p>エストニア政府が複数のエルエルエムを対象に、ロシアの「戦略的ナラティブ」に対する耐性を評価するベンチマーク結果を公開しました。情報戦時代のエーアイ信頼性という新たな評価軸として注目を集めています。</p>
<ul>
<li>エストニア政府が数十のエルエルエムを対象にロシアのプロパガンダ耐性を測定</li>
<li>「戦略的ナラティブ」への同調度合いと批判的反論能力を定量評価</li>
<li>民主主義国家がエーアイを情報防衛インフラとして捉え始めたことを示す先進事例</li>
<li>モデルによって耐性に大きなばらつきがあり、開発元の姿勢が影響している可能性</li>
<li>性能・安全性に加えて「情報戦耐性」がエルエルエム選定の新基準になりつつある</li>
</ul>
<h3>グーグル「クローム スキルズ」──エーアイプロンプトをワンクリックで再利用</h3>
<p>グーグルがクローム向けの新エーアイ機能「スキルズ・イン・クローム」を発表しました。よく使うエーアイプロンプトを保存してワンクリックで呼び出せる機能で、ブラウザ上でのエーアイ活用の効率を高めます。</p>
<ul>
<li>プロンプトをブラウザに保存してワンクリック再利用が可能になるエーアイ拡張機能</li>
<li>毎回手打ちしていた定型的なエーアイ指示の自動化で業務効率向上を狙う</li>
<li>グーグルがジェミニをクローム深く統合してエーアイネイティブブラウザへ進化を加速</li>
<li>個人・企業ユーザーが独自スキルを作成・共有できる可能性もあり、エコシステム拡大に期待</li>
</ul>

<h2>企業・スタートアップ動向</h2>
<h3>アンソロピックIPO準備中──年間収益470億ドルに急成長、ダニエラ・アモダイCOOが強気姿勢</h3>
<p>アンソロピックは2026年5月時点で年間収益が470億ドル（約7兆円）を超えたと発表しました。2025年末の約90億ドルから急成長しており、IPOを前にダニエラ・アモダイCOOはエーアイへの投資収益懸念を一蹴する強気の姿勢を示しています。</p>
<ul>
<li>年間収益が2025年末の約90億ドルから2026年5月には470億ドル超に急拡大</li>
<li>テッククランチが報じたIPO準備段階で、クロードの商業利用拡大が主な成長ドライバー</li>
<li>ダニエラ・アモダイCOOが「エーアイ投資リターンへの懐疑論」を公に否定</li>
<li>企業向けクロードAPIと個人向けクロード.エーアイの両輪で急速な収益拡大を実現</li>
<li>オープンエーアイ・グーグルとの三強競争でアンソロピックが独自ポジションを確立しつつある</li>
</ul>
<h3>メタ、テント型データセンターで建設期間を半減──テスラ方式を援用</h3>
<p>メタが米国オハイオ州ニューオールバニー郊外に「急速展開構造体」と呼ぶテント型データセンター6棟を建設中であることが明らかになりました。テスラのギガファクトリー手法を参考に通常比半分の期間でデータセンターを稼働させる狙いです。</p>
<ul>
<li>オハイオ州に天候対応テント型データセンター6棟を建設中（クリーンビューが確認）</li>
<li>通常の建設工期を半減させる「急速展開構造体」でエーアイインフラを前倒し整備</li>
<li>マーク・ザッカーバーグCEOが「ザ・インフォメーション」インタビューで方針を言及済み</li>
<li>テスラのギガファクトリー・xエーアイも同様のアプローチを採用しており業界トレンドに</li>
<li>エーアイ基盤整備の急ピッチがデータセンター建設業界の新常識を塗り替えつつある</li>
</ul>
<h3>エアビーアンドビー、独自エーアイラボ設立へ──ブライアン・チェスキーCEOが表明</h3>
<p>エアビーアンドビーのブライアン・チェスキーCEOが新たなエーアイラボを立ち上げる計画を明らかにしました。既存のエルエルエムパートナーシップを結んでいない理由を「既存製品はまだ準備不足」と説明、独自開発への意欲を示しました。</p>
<ul>
<li>チェスキーCEOが新エーアイラボ設立の意向をテッククランチに語った</li>
<li>これまでオープンエーアイやアンソロピックとのエルエルエム提携を結んでいなかった</li>
<li>「既存モデルは旅行・ホスピタリティの複雑なユースケースに十分ではない」と判断</li>
<li>エアビーアンドビー独自のデータ（数百万件の宿泊体験）を活かしたエーアイ開発を目指す</li>
<li>大手テック企業以外のプラットフォーム企業が自前エーアイ開発に踏み切る波が到来</li>
</ul>
<h3>アップル、WWDC 2026でシリの大規模刷新を発表予定</h3>
<p>アップルがWWDC 2026（6月9日開幕）でシリの大規模エーアイアップグレードを発表する見通しです。より会話的で文脈を理解できるアシスタントへの刷新が期待されており、アップルインテリジェンスの大幅強化も予告されています。</p>
<ul>
<li>WWDC 2026は日本時間6月10日午前2時からアップル公式チャンネルで配信</li>
<li>シリが「より会話的・文脈理解型」のエーアイアシスタントに大幅進化する見通し</li>
<li>アップルインテリジェンス機能の全面的なアップデートも発表予定</li>
<li>昨年発表したアップルインテリジェンスの実装が遅れていた部分を一気に解消する狙い</li>
<li>グーグル・マイクロソフト・アマゾンのエーアイアシスタントとの真正面からの競争に突入</li>
</ul>
<h3>グーグル、データセンター向けバーチャルパワープラント契約を締結</h3>
<p>グーグルがボルタス社と提携し、米国最大の電力グリッド向けに仮想発電所（バーチャルパワープラント）を構築する契約を締結したとエムアイティーテックレビューが報じました。エーアイデータセンターの電力需要急増に対応する新たなアプローチです。</p>
<ul>
<li>グーグルがボルタス社とバーチャルパワープラントの構築・運用契約を締結</li>
<li>需要応答型の電力管理でデータセンターの電力コストと環境負荷を同時に削減</li>
<li>エーアイ推論のエネルギー消費急増に対応するグリーンインフラ整備の一環</li>
<li>電力使用量削減に応じる一般ユーザーへの報酬プログラムと連動する仕組み</li>
</ul>

<h2>規制・社会影響</h2>
<h3>エーアイ生成訴状が米国裁判所を直撃──判事たちが対応に苦慮</h3>
<p>エーアイを使って作成された訴状や法的文書が米国の連邦裁判所に大量に押し寄せており、判事たちが対処に苦慮しているとエムアイティーテックレビューが報じました。弁護士なしの個人がエーアイで作成した書類を精査する負担が急増しています。</p>
<ul>
<li>コロラド州連邦地裁のマリッツァ・ブラズウェル判事ら複数の判事がエーアイ生成書類問題を訴え</li>
<li>弁護士なし（本人訴訟）の件数が増加し、エーアイ生成文書の質が玉石混交</li>
<li>ハルシネーションを含む架空の判例引用などが訴状に混入するケースが多発</li>
<li>裁判所がエーアイ使用の開示義務ルール整備を急ぐ一方、全国的な統一基準はなし</li>
<li>エーアイが司法アクセスを民主化する可能性と、司法の混乱を招くリスクの両面が浮き彫りに</li>
</ul>
<h3>日本政府、エーアイ「ミトス」のアクセス権を取得──サイバー防衛に活用</h3>
<p>日本政府がエーアイシステム「ミトス（Mythos）」へのアクセス権を取得し、サイバー防衛強化に活用する方針が報じられました。三菱ユーエフジェイ銀行・三井住友銀行・みずほ銀行もアクセス権を取得したとみられています。</p>
<ul>
<li>日本政府が防衛・安保目的でエーアイシステム「ミトス」を導入</li>
<li>メガバンク3行（エムユーエフジェー・エスエムビーシー・みずほ）も同システムへのアクセス取得か</li>
<li>サイバー攻撃の検知・分析・対応にエーアイを活用するインテリジェンス強化の一環</li>
<li>国家安全保障分野でのエーアイ導入が日本でも本格化する先行事例として注目</li>
<li>金融機関がセキュリティエーアイを取得した背景には深刻化するサイバー脅威がある</li>
</ul>
<h3>メルカリ「エーアイ戦国時代」発言──ガバナンス策定の実態を公開</h3>
<p>メルカリがエーアイ統治（ガバナンス）の具体的な取り組みをメディアに公開しました。「この1年はエーアイ戦国時代」と表現しつつ、「シャドーエーアイ」リスクへの対応とエーアイネイティブカンパニーへの転換を両立する戦略を語っています。</p>
<ul>
<li>メルカリが「エーアイ・ネイティブ・カンパニー」宣言後のガバナンス実態を公開</li>
<li>社員がサードパーティエーアイを無断利用する「シャドーエーアイ」リスクへの対応を詳述</li>
<li>エーアイ活用推進とリスク管理を両立する「攻めのガバナンス」の構築事例として注目</li>
<li>日本企業のエーアイガバナンス事例の中でも先進的取り組みとして業界の参考に</li>
</ul>
