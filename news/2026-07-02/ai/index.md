<h2>Cloudflareが「AIはコンテンツ利用料を払え」新方針</h2>
<ul>
<li>Cloudflareが2026年9月15日を期限に、AI企業へ新方針を通告</li>
<li>検索用クローラーとAI学習・エージェント用クローラーを分離登録することを義務化</li>
<li>分離しない、または無許可のAIクローラーは多数のパブリッシャーサイトで既定ブロックに</li>
<li>ニュースサイトなど発信元がAI企業に利用料を求める動きが本格化した形</li>
</ul>
<h2>Anthropic「Fable 5」「Mythos 5」、安全保障審査を経て世界公開</h2>
<ul>
<li>トランプ政権が3週間前、Anthropicの新モデル「Fable 5」「Mythos 5」を国家安全保障上のリスクとして問題視</li>
<li>追加の安全性テストを経て、米国が輸出規制を解除</li>
<li>Anthropicは公式ブログで「Fable 5」を全世界向けに提供開始したと発表</li>
<li>米国内組織は既に一部アクセスを取得しており、フロンティアAIモデルを巡る政府審査の一事例に</li>
</ul>
<h2>LLMは「金太郎飴」思考から抜け出せるか</h2>
<ul>
<li>ChatGPT・Claude・Geminiに「1〜10の乱数を1つ」と尋ねると、多くの場合「7」が返る現象が知られる</li>
<li>大規模言語モデルの出力が特定のパターンに収束しやすい「集団思考（グループシンク）」问題</li>
<li>ある新興企業が、モデルの応答に意図的な多様性を持たせる技術開発に取り組んでいると米メディアが報道</li>
<li>同じ質問への回答が毎回似通うことは、創作支援やブレインストーミング用途での限界として指摘されてきた</li>
</ul>
<h2>プライバシー重視AI「Venice AI」がユニコーン到達</h2>
<ul>
<li>Venice AIが6500万ドルのシリーズAを実施し、企業評価額が10億ドルを突破</li>
<li>年換算売上高（ARR）は既に7000万ドル超で、黒字化も達成済みとCEOのエリック・ボーヒーズ氏が説明</li>
<li>ユーザーデータを収集・保存しない「プライバシーファースト」を掲げるAIプラットフォームとして急成長</li>
<li>大手クラウド系AIサービスとの差別化ポイントとして、データを一切残さない設計を前面に打ち出す</li>
</ul>
<h2>Google「Gemini Spark」がMacに対応</h2>
<ul>
<li>Googleの常時稼働型AIエージェント「Gemini Spark」がMac版の提供を開始</li>
<li>リアルタイムでのタスク追跡機能や、対応アプリの拡充も同時にアップデート</li>
<li>24時間365日ユーザーのタスクを見守り、必要に応じて自律的に動くエージェント型アシスタントとして展開</li>
<li>これまでモバイル・Windows中心だった対応環境がMacへ拡大したことで利用層の拡大を狙う</li>
</ul>
<h2>Meta、AI計算資源の余剰をクラウド事業として販売へ</h2>
<ul>
<li>Metaが自社のAI計算インフラの余剰分を外部に販売する「クラウド事業」の計画を進めていると報じられる</li>
<li>SpaceXも同様に、余剰の計算能力を収益化する動きを見せているという</li>
<li>実現すればAmazon Web Services・Google Cloud・Microsoft Azureなど既存クラウド大手と直接競合することに</li>
<li>巨額投資を続けるAIデータセンターの投資回収策として、計算資源そのものを商品化する動きが広がる</li>
</ul>
<h2>農業のAI活用、データ整備が追いつかず</h2>
<ul>
<li>農業分野でAIの活用機運が高まる一方、業界のデータ基盤整備が追いついていないとの指摘</li>
<li>変動する肥料コストや天候不順、薄い利益率に悩む農業現場でAIによる需要予測などが期待される</li>
<li>AIによる作物予測モデルは収量改善に貢献しうるとの研究結果がある一方、基盤データの質・量が課題に</li>
<li>導入を急ぐ前に、データ整備を先行させる必要があると業界関係者が警鐘を鳴らす</li>
</ul>
<h2>中国、2682光子の光量子コンピュータ「天衍-P2000」をクラウド公開</h2>
<ul>
<li>中国電信（チャイナテレコム）系の量子技術グループが、光量子コンピュータ「天衍（Tianyan）-P2000」を稼働開始</li>
<li>2,682個の光子を同時に制御・追跡できる干渉計型導波路メッシュを採用</li>
<li>中国科学院の量子情報研究拠点と共同開発した「九章4.0」の技術を基盤に構築</li>
<li>ある複雑な計算をわずか29マイクロ秒で完了。従来型スーパーコンピュータでは160億年かかる規模と試算</li>
<li>常温動作が可能で、超伝導方式のような極低温冷却が不要な点がコスト・拡張性の強みに</li>
<li>光方式・超伝導方式の両方で量子優位性サービスを提供する、世界初のマルチテナント基盤とされる</li>
</ul>
