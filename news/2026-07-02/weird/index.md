<h2>世界一高額な「手紙」、中身は驚くほど退屈だった件</h2>
<ul>
<li>1993年のオークションで超高額落札された「ボルドー・レター」は、1847年にモーリシャス島の港町ポートルイスのワイン商人エドワード・フランシスが、フランス・ボルドーの取引相手に送った業務連絡の手紙</li>
<li>内容はワイン樽48樽を受け取り、すでに約3分の1を売却済みと伝えるだけの、極めて事務的で退屈な文面</li>
<li>高額な理由は中身ではなく貼られた切手。印刷ミスで「Post Paid（郵便料金支払済）」と刷るべきところを「Post Office」と誤植した「モーリシャス・ブルー」と「モーリシャス・ピンク」の希少切手2枚が貼付されていたため</li>
<li>ミス版切手は500枚刷られたのち訂正され、現存するのは青12枚・ピンク15枚の計27枚のみ。このレターの推定価値は500万ドル以上、単体のモーリシャス・ブルーなら1,000万〜1,700万ドルとも言われる</li>
<li>現在の所有者はシンガポール在住の匿名コレクターで、1993年から非公開のまま保管している</li>
<li>出典: <a href="https://www.odditycentral.com/news/the-worlds-most-expensive-letter-is-surprisingly-boring.html">Oddity Central</a></li>
</ul>
<h2>ゴキブリに「潜水服」を開発、シンガポールの研究チームが災害救助用に実用化</h2>
<ul>
<li>シンガポール南洋工科大学のヒロタカ・サト教授らのチームが、マダガスカルオオゴキブリ向けの3Dプリント製ミニ潜水服を開発</li>
<li>素材はレジン樹脂製。過酸化水素と二酸化マンガンを反応させて酸素を発生させ、ゴキブリの呼吸器官「気門」に供給する仕組みで、水中でも最大3時間の呼吸を可能にする</li>
<li>潜水服を着けたゴキブリの水中移動速度は秒速78.4ミリメートルで、陸上時より少し遅くなる程度</li>
<li>そもそもの目的は、同チームが開発を進める遠隔操作型「ゴキブリ・ドローン」の弱点だった「水に弱い」問題の克服。これにより水没した瓦礫の中の捜索・救助活動への応用が期待される</li>
<li>実際に2025年春のミャンマー地震の救助活動「ライオンハート作戦」でも投入された実績がある</li>
<li>出典: <a href="https://www.odditycentral.com/news/scientists-develop-cockroach-diving-suits-for-the-benefit-of-mankind.html">Oddity Central</a></li>
</ul>
<h2>重機なし・ノミとハンマーだけ、中国の村人80人が15年がかりで断崖に道を彫った</h2>
<ul>
<li>中国南西部・重慶市の標高約914メートルの山中にある「藍鷹（ランイン）断崖道路」は、地元ランイン村の23世帯・約80人の村人が力を合わせて作り上げた全長1.2キロの手彫り道路</li>
<li>重機が入れない断崖の狭いスペースだったため、作業はすべてノミとハンマーによる手作業。各世帯が道の区間を分担して掘り進めた</li>
<li>2001年に着工し基礎部分の完成までに約4年、その後も工事が続き2016年に地元当局が補強・拡幅を承認</li>
<li>道幅は極めて狭く、区間によっては傾斜22.9%に達する箇所もあり、「世界で最も危険な道路の一つ」とも評される</li>
<li>建設の背景には、外界からほぼ完全に孤立していた村の貧困緩和という切実な目的があった。今では断崖から望む大渓谷の絶景スポットとしても知られる</li>
<li>出典: <a href="https://www.odditycentral.com/travel/chinas-breathtaking-lanying-cliff-road-was-hand-carved-through-the-side-of-a-mountain.html">Oddity Central</a></li>
</ul>
<h2>「いつもの45ドルだよ」と嘘をついた芝刈りバイト少年、感謝のチップを定価と勘違いされ解雇</h2>
<ul>
<li>ある家主が近所の少年に庭の芝刈りを依頼しており、料金は前庭・裏庭・雑草取り込みで40ドルという取り決めだった</li>
<li>少年の仕事ぶりが良かったため、家主は5月のある回に感謝の気持ちとして5ドルのチップを上乗せして支払った</li>
<li>ところがその2週間後の次の依頼で、少年は「いつもの料金は45ドルだよ」と、チップ込みの金額をあたかも正規料金であるかのように請求</li>
<li>家主が過去の取引履歴を示して指摘すると、少年は言い訳を重ねて値上げを正当化しようとした</li>
<li>結局家主は少年への依頼をやめ、持病を抱えながらも自分で芝刈りをする道を選んだ。「駆け引きは経験不足からくるもので悪意はなさそうだが、痛い授業料だったはず」とネット上でも話題に</li>

</ul>
