<h2>新モデル・研究トピック</h2>
<h3>OpenAI、チャットジーピーティーに「ロックダウンモード」を導入——プロンプトインジェクション対策</h3>
<p>オープンエーアイがチャットジーピーティーに新たなセキュリティ機能「ロックダウンモード」を追加した。有効にするとウェブブラウジングや外部サービスへの接続が制限され、プロンプトインジェクション攻撃による機密データ漏洩リスクを大幅に低減できる。医療・法務・金融など機密情報を扱う組織向けの機能として位置付けられている。</p>
<ul>
<li>プロンプトインジェクションとは、AIへの指示に悪意ある命令を混入させ誤動作させる攻撃手法</li>
<li>ロックダウンモード有効時はWebブラウジング・コードインタープリター・外部API連携が制限される</li>
<li>昨日報告されたメタのAIエージェント乗っ取り事件を受けて開発が加速したとみられる</li>
</ul>
<h3>ケンブリッジ大学、AIが設計した"万能ワクチン"の臨床試験に成功</h3>
<p>英ケンブリッジ大学が、エーアイが設計した抗原を用いる「ユニバーサルワクチン」の初の臨床試験成功を発表した。コロナウイルスの親戚にあたるサルベコウイルス群のゲノム配列を機械学習で解析し、グループ共通の「スーパー抗原」を設計。健康な39人への投与で安全性と免疫応答を確認した。既知の変異株だけでなく、未知の変異株にも効果が期待される画期的なアプローチ。</p>
<ul>
<li>従来のワクチンは既知のウイルスにしか対応できないが、本手法は未知の変異にも備えられる</li>
<li>エーアイによるゲノム解析が医薬品設計に本格的に貢献し始めた象徴的な成果</li>
<li>新型コロナ以外のウイルスへの応用も視野に入れた研究が続く</li>
</ul>
<h3>WWDC 2026でシリが大幅刷新——アップルインテリジェンス新機能も</h3>
<p>アップルの年次開発者会議「WWDC 2026」が今週月曜に開幕し、シリの抜本的な機能強化とアップルインテリジェンスの新機能が発表予定。昨年の不評を受けて大幅刷新されたシリは、より自然な会話と高度なタスク実行能力を備えたとされ、業界の注目を集めている。</p>
<ul>
<li>シリはサードパーティアプリとの連携強化と、より文脈を理解した応答が可能になる見込み</li>
<li>アップルインテリジェンスの日本語対応拡充も発表される可能性が高い</li>
<li>グーグルやオープンエーアイとの差別化として「プライバシー優先のオンデバイスエーアイ」を強調</li>
</ul>
<hr />
<h2>企業・スタートアップ動向</h2>
<h3>トランプ政権、オープンエーアイへの政府出資を検討——「アメリカ国民が利益を享受すべき」</h3>
<p>トランプ大統領がオープンエーアイへの政府出資を検討していることが明らかになった。「エーアイの成功からアメリカ国民が利益を得られる仕組みを議論している」と述べ、政府がエーアイ企業の株式を取得するという前例のない措置を示唆した。オープンエーアイは非営利法人から株式会社への転換を進めており、このタイミングでの政府関与は業界に波紋を広げている。</p>
<ul>
<li>政府が民間テクノロジー企業の株式を保有するのは米国では極めて異例</li>
<li>オープンエーアイの企業転換交渉の行方に政府出資という新たな変数が加わった形</li>
<li>「エーアイを国家戦略資産として管理したい」というトランプ政権の意向が鮮明に</li>
</ul>
<h3>ホワイトハウスのエーアイ政策責任者が退任——後継の政策機関設立へ</h3>
<p>ホワイトハウスのエーアイ政策顧問を務めていたスリラム・クリシュナン氏が退任することが明らかになった。元マイクロソフト・ツイッターの幹部であるクリシュナン氏は今後、トランプ政権のエーアイ政策形成を側面から支援する新機関を立ち上げる予定。政府内のエーアイ政策リーダーシップの移行期に入ったとみられる。</p>
<ul>
<li>クリシュナン氏はシリコンバレーとワシントンの橋渡し役として、OSTP（科学技術政策局）主任エーアイ顧問として活動</li>
<li>トランプ政権はエーアイを「国家競争力の核心」と位置付けており、政策の継続性が重視される</li>
<li>後継の枠組みが明確になるまでの政策的空白期が懸念される</li>
</ul>
<h3>住友ゴム・富士通、エーアイでタイヤ解析時間を45分から5分に短縮</h3>
<p>住友ゴム工業と富士通がエーアイを用いた「サロゲートモデル」を共同開発し、タイヤの性能予測解析時間を従来の45分から約5分に短縮することに成功した。約60万要素規模の複雑なシミュレーションをエーアイが代替する手法で、製造業のエーアイ活用の先進事例となっている。</p>
<ul>
<li>従来はスーパーコンピュータで数十分かかるタイヤ変形解析をエーアイが高速予測</li>
<li>製品開発サイクルの大幅短縮と設計コスト削減が期待される</li>
<li>自動車・製造業全般でのデジタルツイン×エーアイ活用が加速する先行事例</li>
</ul>
<hr />
<h2>規制・社会影響</h2>
<h3>AIが生成した訴状が米国の裁判所に氾濫——裁判官たちが対処に苦慮</h3>
<p>アメリカの連邦裁判所で、エーアイが生成したとみられる訴状の提出が急増していることが報告された。コロラド州の連邦判事は「毎日大量の素人作成文書を読んでいるが、エーアイ生成のものは誤字脱字がなくても事実誤認やでたらめな法的根拠が含まれている」と語った。弁護士費用を払えない当事者がエーアイを頼るケースが急増しており、司法制度の課題として浮上している。</p>
<ul>
<li>エーアイが生成する文書は文法的に正確でも、法的に成立しない主張や誤った判例引用を含む</li>
<li>弁護士なしで訴訟する「本人申立」案件にエーアイが多用されており、裁判所の業務量が増加</li>
<li>米国の複数の裁判所がエーアイ使用の開示義務を設けるルール整備を急いでいる</li>
</ul>
<h3>日本漫画家協会、エーアイ学習禁止の契約条項を公表——著作物保護のガイドライン</h3>
<p>日本漫画家協会が、自作の著作物をエーアイに学習・改変されることを防ぐための契約条項例を公表した。「納品後にエーアイ学習に利用しない」「エーアイで画像を改変しない」といった文言を盛り込んだ契約書サンプルを提供し、クリエイターが自衛できる環境整備を支援する取り組み。</p>
<ul>
<li>エーアイ学習に対する著作権の法的保護が日本ではまだ不明確な部分が多い</li>
<li>業界団体が自衛のための具体的な契約文言を示した先進的な取り組みとして注目される</li>
<li>イラストレーター・写真家・音楽家など他クリエイター業界にも波及効果が期待される</li>
</ul>
